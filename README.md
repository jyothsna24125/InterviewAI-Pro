# InterviewAI-Pro

An AI-powered interview preparation platform that helps users practice technical interviews through AI-generated questions, voice interaction, and intelligent feedback.

## ✨ Features
- AI-generated interview questions
- Voice input support
- Webcam integration
- Real-time interview timer
- AI-based evaluation and feedback
- Responsive user interface

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Google Gemini API
- Git & GitHub

## 🚀 Getting Started

```bash
git clone https://github.com/jyothsna24125/InterviewAI-Pro.git
cd InterviewAI-Pro
npm install
npm start
```
